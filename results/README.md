# Results Directory

This directory contains the analytical results of the research project.

## Files
- `model_comparison.csv`: Performance metrics for FP32 baselines.
- `compression_results.csv`: Step-by-step impact of model optimization (Quantization and Pruning).

## Key Finding
The final model achieved **92.3% accuracy** at only **4.5 MB**, meeting all edge deployment constraints for the construction site use-case.
